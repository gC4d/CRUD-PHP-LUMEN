Apresentação da API

GET /api/users

Descrição: Retorna uma lista de todos os usuários.

POST /api/users

Descrição: Cria um novo usuário.

GET /api/users/{id}

Descrição: Retorna detalhes de um usuário específico identificado pelo ID.

PUT /api/users/{id}

Descrição: Atualiza as informações de um usuário específico identificado pelo ID.

DELETE /api/users/{id}

Descrição: Remove um usuário específico identificado pelo ID.
